# Länderquiz

A geography quiz game.

You can choose between a country, a capital and a vexillology/flag quiz.

## Usage

Start the game by clicking on länderquiz.exe.

Choose one of the different quizzes, type in the correct answer 
and press enter/return. You'll see the percentage of correctly 
answered questions in the top-right corner. Press ESC to go 
back to the main menu.

## Requirements

Python 3.4 and above.
